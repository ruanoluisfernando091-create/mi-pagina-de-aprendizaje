Fundamentos de la programación

¿Qué es el pensamiento lógico? ¿Cómo piensa un programador, para resolver un problema?

El pensamiento lógico (a veces llamado pensamiento computacional cuando se refiere a programadores) es la forma de analizar y resolver problemas paso a paso, siguiendo reglas claras, ordenadas y coherentes. Es la base de cómo piensa un programador al enfrentarse a cualquier situación.

Un programador no solo piensa en la respuesta final, sino en cómo llegar a ella de manera clara, eficiente y sin ambigüedades. Te lo explico en partes:

Características del pensamiento lógico de un programador:

  • Descomposición: Dividir un problema grande en partes más pequeñas y manejables.
  • Secuencia: Ordenar los pasos de manera lógica, como una receta de cocina.
  • Condicionales: Pensar en opciones y caminos alternativos.
  • Abstracción: Ignorar lo que no importa y enfocarse en lo esencial.
  • Reutilización y patrones: Usar soluciones que ya funcionan para problemas similares.
  • Pensamiento algorítmico: Traducir la solución a pasos exactos que una computadora pueda ejecutar.

¿Qué son los tipos de datos?

Imagina que la computadora es como una bodega 🏠 con cajas para guardar cosas. Cada caja está hecha para guardar un tipo de objeto:

  • 📦 Número entero (int) → para guardar números sin coma: 10, -3, 1000
  • 📦 Número con coma (float) → para guardar decimales: 3.14, 2.5, -0.75
  • 📦 Texto (string) → para guardar palabras o frases: "Hola", "Luis"
  • 📦 Booleano (bool) → para guardar solo dos opciones: Verdadero (True) o Falso (False)
  • 📦 Carácter (char) → para una sola letra: 'A', 'B', '9'
  • 📦 Lista (array / list) → para guardar varios datos juntos: [1, 2, 3]

👉 Los tipos de datos son las formas en que la computadora sabe qué tipo de información guardas en una "caja".

¿Qué son los conceptos básicos?

Son como las reglas del juego para que la computadora entienda qué hacer con esos datos.

  • Variable: Una caja con nombre para guardar datos.
  • Constante: Una caja que no cambia.
  • Operadores: Símbolos para hacer operaciones.
  • Condicional: Es una decisión: si pasa algo → haz esto, si no → haz otra cosa.
  • Bucle (loop): Repite algo varias veces.
  • Función: Una receta: conjunto de pasos que puedes usar varias veces.

Explicaciones de Datos

1. Entero (int)

Los enteros son números completos, es decir, que no tienen decimales ni parte fraccionaria. Ejemplos: 0, 1, 25, -3, 1000.

2. Decimal (float)

Los decimales (o flotantes) son números que sí tienen parte decimal. En programación se escriben con punto (no con coma). Ejemplos: 3.14, 2.5, -0.75.

3. Texto (string)

Los strings son cadenas de texto: letras, números, espacios o símbolos. Siempre se escriben entre comillas (simples ' o dobles "). Ejemplos: "Luis", "Hola", "123".

4. Booleano (bool)

Un booleano solo puede tener dos valores: True (verdadero) y False (falso). Se usan para representar respuestas lógicas: sí/no.

5. Lista (list)

Las listas son conjuntos de datos ordenados y agrupados dentro de corchetes []. Pueden contener números, textos, booleanos o incluso otras listas. Ejemplos: [1, 2, 3], ["pera", "uva", "manzana"].
